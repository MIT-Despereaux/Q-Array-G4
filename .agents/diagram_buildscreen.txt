PARAMETER MEANINGS
  screen_radius    inner radius of the cylindrical wall (CylWall)
  screen_e         wall thickness  (= "se" above)
  screen_bot_h     height of bottom wall section; FlangeBot sits at its top
  screen_mid_h     height of middle wall section; FlangeMid sits at its top
  screen_top_h     height of top    wall section; FlangeTop sits at its top
  flange_e         how far each ring flange protrudes BEYOND the outer wall
  flange_h         FlangeTop height = flange_h; FlangeBot/Mid height = 2*flange_h

  position         AUTO-DERIVED -- never set manually:
                   position = plate_z_bottom - bot_h - mid_h - top_h

  flange_e per screen is also derived:
                   flange_e = PlateXK_radius - ScreenXK_radius - ScreenXK_e

MATERIAL NAMING (from BuildScreen source)
  Copper screens:
    CylWall  -> "Copper" + name + "Wall"       e.g. "CopperScreen1KWall"
    Bottom   -> "Copper" + name + "Bottom"
    FlangeBot-> "Copper" + name + "FlangeBottom"
    FlangeMid-> "Copper" + name + "FlangeMid"
    FlangeTop-> "Copper" + name + "FlangeTop"
  OVC (stainless, triggered when mat contains "StainlessSteel"):
    CylWall  -> "StainlessSteelOVCWall"
    Bottom   -> "StainlessSteelOVCBottom"
    All flanges -> "StainlessSteelOVCFlange"
