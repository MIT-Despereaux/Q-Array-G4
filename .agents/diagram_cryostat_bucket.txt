================================================================================
  Diagram A: Bucket shape anatomy
  (right-half cross-section, axis on left, +z upward)
================================================================================

  Bucket(name, innerRadius, thickness, height)
  Origin = inner bottom center  (junction between bottom disk and wall)

  <- innerRadius ->  th
                     ick

  |                  |##|         <- top rim, flush with anchoring plate bottom
  |                  |  |            z = position + height
  |                  |  |
  |                  |  |
  |      (hollow)    |  |         <- cylindrical wall
  |                  |  |            ri = innerRadius
  |                  |  |            ro = innerRadius + thickness
  |                  |  |
  |                  |  |
  |                  |  |         <- z = position  (inner bottom center = origin)
  |##################|##|         <- bottom disk (solid, ri = 0)
  |##################|##|            thickness = "thickness" param (same as wall)
  =====================            z = position - thickness

  PARAMETER MEANINGS
    innerRadius   inner radius of the hollow cylindrical wall
    thickness     wall thickness = bottom disk thickness (same value for both)
    height        height of the cylindrical wall section (NOT including bottom disk)
    position      z coordinate of the inner bottom center (the origin of this shape)
                  -> wall spans  z = position      to  z = position + height
                  -> disk spans  z = position-thickness  to  z = position

  DESIGN RULE (from Screen_h() formulas in cache):
    Each screen's TOP RIM reaches exactly to the BOTTOM FACE of its anchoring plate.
    So:   position + height = z_bottom_of_anchor_plate
    i.e.: height = (z_bottom_of_anchor_plate) - position

================================================================================
  Diagram C: Bucket cascade -- position and height derivation for DSPX
  (no Screen10mK; z=0 at Plate10mK bottom face)
================================================================================

  Plates stack upward (+z):

    z_Plate10mK_bottom  = 0                            (definition)
    z_Plate10mK_top     = Plate10mK_e = 5 mm

    z_Plate100mK_bottom = Plate10mK_e + DisPlate10mK_100mK
                        = 5mm + 160mm = 165 mm
    z_Plate1K_bottom    = z_Plate100mK_bottom + Plate100mK_e + DisPlate100mK_1K
                        = 165mm + 4mm + 94mm = 263 mm
    z_Plate4K_bottom    = z_Plate1K_bottom + Plate1K_e + DisPlate1K_4K
                        = 263mm + 4mm + 203.5mm = 470.5 mm
    z_Plate50K_bottom   = z_Plate4K_bottom + Plate4K_e + DisPlate4K_50K
                        = 470.5mm + 4mm + 203.2mm = 677.7 mm

  Screens hang downward (-z) via cascade of gap parameters:

    position_Screen1K  = -DisPlate10mK_BottomScreen1K          <- [TBD from DSPX drawings]
    height_Screen1K    = z_Plate1K_bottom - position_Screen1K  <- derived automatically
                       = 263mm + DisPlate10mK_BottomScreen1K

    position_Screen4K  = position_Screen1K - Screen1K_e - DisOuterBottomScreen1K_InnerBottomScreen4K
                       = position_Screen1K - 2mm - 19.5mm      <- gap = 1.95cm from CROSS
    height_Screen4K    = z_Plate4K_bottom - position_Screen4K  <- derived

    position_Screen50K = position_Screen4K - Screen4K_e - DisOuterBottomScreen4K_InnerBottomScreen50K
                       = position_Screen4K - 2mm - 19.8mm      <- gap = 1.98cm from CROSS
    height_Screen50K   = z_Plate50K_bottom - position_Screen50K <- derived

    position_OVC       = position_Screen50K - Screen50K_e - DisOuterBottomScreen50K_InnerBottomOVC
                       = position_Screen50K - 2mm - 20mm       <- gap = 2.0cm from CROSS
    height_OVC         = ?                  <- total OVC height, sets how far up OVC reaches
                                               (must clear Plate50K + leave room for Bride)

  Bride (ReversedBucket):
    position_Bride     = z_Plate50K_bottom + DisPlate50K_Bride + Bride_h
                       = 677.7mm + 126mm + 111mm = 914.7mm  (all from CryoConcept2020)
